# Sales Data Analysis (Python Project)

## 📊 Project Overview
Analyzed 20K+ sales records using Python (Pandas) to identify revenue trends, product performance, and customer behavior patterns.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Excel (Data Cleaning)
- Jupyter Notebook

## 🔍 Key Insights
- Top 3 products contribute ~60% of total revenue
- Region A shows highest sales growth
- Seasonal trends impact overall sales performance

## 📈 Outcome
Provided actionable insights to improve product strategy, customer targeting, and revenue growth.
